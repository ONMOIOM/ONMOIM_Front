## 설명 (Description) — 아래 내용 복사해 PR 본문에 붙여넣기

```markdown
## 개요
행사 목록 갱신, 참여자/댓글 UX, 프로필 메뉴 정리, API 스펙 반영 등 메인·행사 상세·프로필 관련 개선을 반영했습니다.

## 변경 사항

### 행사 목록·캐시
- **발행/수정 후 목록 갱신**: 행사 발행(저장) 또는 행사 상세에서 수정 후 메인 이동 시 `eventList`, `myHostedEvents`, `myParticipatedEvents` 무효화 및 refetch 후 이동하도록 수정
- **행사 카드/추가 카드 클릭 시**: 상세·수정·추가 페이지로 이동할 때 위 쿼리 무효화 → 복귀 시 목록 리로딩
- **홈 쿼리**: `refetchOnMount: true` 적용으로 탭 복귀 시 stale이면 자동 리패치
- **삭제 시 모든 탭 반영**: 행사 삭제 시 setQueryData로 해당 eventId 제거 + invalidate로 동기화하여 모든 탭에서 즉시 반영

### 행사 상세(EventPost)
- **참여 상태 변경 시**: 참여/고민중/못가요 선택 시 참여자 목록 refetch + 홈용 쿼리 무효화
- **참여자 표시**: 참여(ATTEND)만 카드·인원 수에 반영, 참여자/모달 이미지는 API `imageUrl` fallback 적용
- **댓글 삭제**: 본인 댓글에만 삭제 버튼 노출, `DELETE /api/v1/comments/{commentId}` 연동 및 삭제 후 목록 갱신
- **수정 버튼 클릭 시**: 저장 후 목록 refetch 후 메인으로 이동 (썸네일 등 최신 데이터 반영)

### 행사 생성·수정
- **PATCH 스펙**: `schedule`/`location` 중첩 제거, flat 필드(`startTime`/`endTime`, `streetAddress`/`lotNumberAddress`)로 전송
- **날짜/시간 모달**: 시간대 선택 후 `setActive(null)` 제거하여 같은 박스로 연속 선택 가능하도록 수정

### 일주일 이내 탭
- **기간 계산**: 오늘 00:00 ~ 7일 후 23:59 구간으로 필터링하도록 수정하여 해당 기간 행사 카드 정상 노출

### 프로필 메뉴
- **문의하기 제거**: 문의하기 메뉴 항목 및 관련 아이콘 제거
- **로그아웃 위치**: 같은 행사 참여자 보기 아래로 이동 (1번: 같은 행사 참여자 보기, 2번: 로그아웃)
- **모달 높이**: 문의하기 제거에 맞춰 세로 높이 축소 (`h-[413px]` → `h-[355px]`), 하단 패딩 `py-8` → `py-6`

### 기타
- **같은 행사 참여자 보기 페이지**: 참여자 이미지를 메인과 동일하게 `getUserProfile`로 보강, 카드 클릭 시 프로필 이동 유지
- **프로필 스크롤**: 프로필 페이지 전환 시 맨 위로 스크롤
- **홈 행사 카드 이미지**: 모든 탭(행사 목록, 일주일 이내, 내가 여는 행사, 참여한 행사)에서 행사 썸네일 공통 표시 (`eventIdToImageUrl` 활용)

## 테스트
- [ ] 행사 발행 후 메인에서 즉시 목록 반영
- [ ] 행사 상세에서 수정 후 메인 복귀 시 목록·썸네일 갱신
- [ ] 일주일 이내 탭에서 기간 내 행사 노출
- [ ] 참여/고민중/못가요 변경 시 참여자 목록·모달 반영
- [ ] 본인 댓글 삭제 동작 및 목록 갱신
- [ ] 프로필 메뉴에서 문의하기 없음, 로그아웃 위치·모달 높이 확인
```
